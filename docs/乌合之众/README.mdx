《乌合之众：大众心理研究》是法国社会心理学家古斯塔夫·勒庞于1895年出版的经典著作，被誉为群体心理学的奠基之作。勒庞在书中深入剖析了群体行为的本质，认为个体一旦融入群体，便丧失了原有的理性和批判精神，转而表现出冲动、情绪化、易怒和智力低下等特征。他指出，群体极易受到暗示和引导，群体的行为远比个体更容易极端，对社会和历史发展产生巨大影响。

勒庞结合自己在动荡年代的亲身经历，通过对群体道德、情感、想象力和信念等方面的分析，揭示了群体心理在政治、法律以及特殊社团中的作用。他也强调，群体既有可能走向集体犯罪，也能够被正能量激发出英雄主义壮举。本书不仅影响了后来的心理学与政治学，还深刻启示了人们理解集体行为与社会变革的规律，被视为群体行为研究者不可不读的佳作。
