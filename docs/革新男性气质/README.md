《Des hommes justes: Du patriarcat aux nouvelles masculinités》（译作“正义的男人：从父权制到新男性气质”或“革新男性气质”）是法国历史学家Ivan Jablonka的重要著作。本书基于对父权制历史的全面梳理，提出对男性身份的新思考，主张男性应积极参与性别正义，以权利平等为核心，重新定义“好男人”的标准及新的男性气质。
