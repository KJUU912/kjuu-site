《鞋狗》是耐克创始人菲尔·奈特的亲笔自传，真实再现了耐克从无到有的创业历程。菲尔·奈特以朴实、幽默的语言，叙述了自己作为一个普通的俄勒冈州人，因热爱跑步而投身于跑鞋销售，从50美元起步，带领一支性格各异的团队，历经银行拒贷、资金短缺、合作伙伴背叛等重重考验，最终缔造出闻名世界的运动品牌。

本书着重展现了创业过程中面对困境的坚持与激情，以及“鞋狗”们对理想的忠诚和疯狂。它不仅是一部商业传记，更是一段跌宕起伏、充满冒险与情感纠葛的真实人生故事，为所有怀揣梦想的人带来启示与鼓励。
