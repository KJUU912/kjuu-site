:::info[期权（Option）]

<Tabs>
<TabItem value="a" label="基本定义">
期权是一种金融衍生品合约。它赋予持有人一个**权利（但不是义务）**：在未来某个时间点或之前，以约定好的价格（**行权价，Strike Price**）买入或卖出某种资产（通常是股票）。买方需要为这种权利支付一笔费用，叫做**期权费（Premium）**。

期权可以用于：
- **投机**：用较小的资金博取更大收益。
- **对冲**：像保险一样，为持有的资产提供价格保护。
</TabItem>
<TabItem value="c" label="生活类比：房产定金">
假设你看中一套房，价格100万。中介说：先交1万定金，未来6个月你有权利用100万买下。
- 如果房价涨到120万，你用100万买到，立刻赚20万。
- 如果房价跌到80万，你可以选择不买，最多损失那1万定金。

这里：
- 定金 = 期权费（Premium）
- 100万 = 行权价（Strike Price）
</TabItem>
<TabItem value="d" label="期权费的构成">
期权费（Premium）= 内在价值 + 时间价值：
- **内在价值（Intrinsic Value）**：如果现在立刻行权能赚到的钱。
- **时间价值（Time Value）**：市场为“未来可能赚钱”的机会所支付的溢价。

到期日时，时间价值衰减为零，仅剩内在价值。

看跌期权内在价值为 $max⁡(K−S, 0)$；看涨为 $max⁡(S−K, 0)$
- K（执行价 / Strike Price）：这是期权合约中约定好的价格
- S（现价 / Spot Price）：这是标的资产当前的市场价格。
</TabItem>
<TabItem value="b" label="看涨或看跌">
- **看涨期权（Call Option）**：赋予你按行权价买入资产的权利。适合看多，赌价格会上涨。
  - 现价 = 100，执行价 = 90，内在价值 = 10（**实值/价内**，In the Money，ITM），期权费最高（等于内在价值+时间价值）
  - 现价 = 100，执行价 = 100，内在价值 = 0（**平值**，At the Money，ATM），期权费居中（等于适中的时间价值）
  - 现价 = 100，执行价 = 110，内在价值 = 0（**虚值/价外**，Out the Money，OTM），期权费最低（等于较低的时间价值）
- **看跌期权（Put Option）**：赋予你按行权价卖出资产的权利。适合看空，赌价格会下跌。
  - 现价 = 100，执行价 = 110，内在价值 = 10（**实值/价内**，In the Money，ITM），期权费最高（等于内在价值+时间价值）
  - 现价 = 100，执行价 = 100，内在价值 = 0（**平值**，At the Money，ATM），期权费居中（等于适中的时间价值）
  - 现价 = 100，执行价 = 90，内在价值 = 0（**虚值/价外**，Out the Money，OTM），期权费最低（等于较低的时间价值）
</TabItem>
</Tabs>
